# Contributing

This project is not a "official" open-source codebase. It is a modified build of a Minecraft mod updated at the bytecode level to work with Twitch API changes.

Pull requests are not accepted because changes cannot be fully verified for safety.

## Why contributions are disabled

- The project is not a fully open-source codebase
- Modifications are made at the compiled (JAR/bytecode) level, which makes safe review of external changes difficult

## Fix requests

If you need support for a specific Minecraft version or encounter issues, feel free to open an issue. I may implement additional version support when I have time.
